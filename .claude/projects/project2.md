# 專案二：房仲內部物件管理系統

純內部使用，不對外公開。

### 五層架構
公司聯盟 → 公司團隊 → 小組 → 個人專區 → 物件
物件可選擇性往上推送共享

### 權限設計
- 聯盟層：各公司老闆是管理員，可管理其他公司物件
- 每層只看得到自己權限內的物件

### 核心功能
- 物件新增/編輯/刪除
- 照片上傳（Supabase Storage）
- 影片上傳（Supabase Storage）
- 搜尋篩選
- 推送到租屋平台一鍵發布
- LINE Bot 快速上架

### 聯賣配件系統
- 房仲身份驗證後上傳名片開啟配件功能
- 可提出聯賣配件請求
- 選擇性推送至小組/公司/聯盟各層級
- 成交後拆傭分成

### 後期規劃
- AI 代理人用 Computer Use 自動操作 LINE
- 抓取記事本文字/照片/影片整理上傳
- 串接惡房東/租霸驗證平台信用評分

### MVP 目標（當兵前完成）
- 物件新增/編輯
- 照片影片上傳
- 基本搜尋
- 推送到租屋平台
